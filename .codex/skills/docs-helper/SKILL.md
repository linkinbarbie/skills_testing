# Docs Helper

## Overview

Summarize README.md and SKILLS.md in plain, short English. Point to the exact files that contain the answer.

## Quick Start

- Open `README.md` and `SKILLS.md`.
- Give a short summary in simple English.
- Answer the specific question and cite the file path you used.
- If the user wants to test a skill, list the steps and the exact prompt to trigger it.

## Tasks

### Summarize Repo Docs

Summarize the key sections of `README.md` and `SKILLS.md`. Keep it short and clear.

### Explain SKILLS.md And Skill Triggers

Explain how skills are listed and how a skill is triggered. Point to `SKILLS.md` and the referenced `SKILL.md` file.

### Locate Skill Files

Tell the user where a skill lives under `.codex/skills/<skill-name>/` and point to its `SKILL.md`.
